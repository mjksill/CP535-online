Curves in space

We parameterize a curve by giving the position of all the points on the curve as a function of some parameter \(u\): \({\bf R}(u)\). We can think of the parameter \(u\) as time. If we do this, we can then consider the derivative of the position of a point on the curve with respect to \(u\) as a velocity:

\begin{align*} \frac{d{\bf R}(u)}{du} \end{align*}

tangent vector: this is a unit vector pointing along the curve. This is simply the normalized "velocity".

arc length: A key quantity of a curve is the length it sweeps out between any two points. We can determine this by using the Pythagoras theorem, along with some calculus.

\begin{align*} ds^2 = dx^2 + dy^2 \end{align*}

where \(ds\) is a bit of length along the arc. Note that this is not really along the arc, but the smaller dx and dy are, the closer it will be.

\begin{align*} ds^2 = dx^2 + dy^2 \\ \left(\frac{ds}{du}\right)^2 &= \frac{d{\bf R}(u)}{du}\cdot\frac{d{\bf R}(u)}{du} \end{align*}

Capillary bridges

If you ever built a sand castle on the beach, you might have noticed that you need to get the sand a bit wet in order to get it to stick and hold a shape. There should be enough water to ensure that all the gains of sand are wet, but not so much that it swamps the sand, washing it away. Have you ever wondered why the grains of sand can be made to stick together to form different shapes?

We will consider the capillary force between two planar surfaces, between a sphere and a planar surface, and between two spherical particles.
